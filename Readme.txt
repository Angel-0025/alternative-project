## Getting Started

These instructions will get you a copy of the project up and running on your local host for testing purposes. See deployment for notes on how to deploy the project.

## Prerequisites

What things you need to install to make the system work in your computer.

- PHP Version 8.1.5
- XAMPP Version 3.3.0
- VSCode Version 1.69.1
- Phpmyadmin Version 5.2.0


## Installation

To make the project work:

- Download and extract the project system on your computer.
- Open the Xampp Control Panel and start the Module of Apache and MySQL.
- Open the system file with Visual Studio Code.

To use the database:
- Go to your computer web browser, and type the url "http://localhost/phpmyadmin/".
- Click the "New" and name the new database "alternative_project".
- Click the import in the navigation bar, and choose the sql file inside the database folder then click the import button.

To customize the email sender:
- Follow the tutorial in this website "https://www.codingnepalweb.com/configure-xampp-to-send-mail-from-localhost/".
